Everything your community needs

All the features you love from Discord, plus the business tools you've been missing.

💬

Rich Text Messaging

Markdown, reactions, replies, threads, file uploads, embeds, and link previews. Everything you'd expect from a modern chat.

🎙️

Voice & Video

Persistent voice channels powered by LiveKit. Camera, screen share, and adaptive quality. Just click and talk.

📋

Kanban Boards

In-channel task management with drag-and-drop. Assign, prioritize, and track — without leaving the conversation.

Unique to Valhalla
📖

Wiki / Knowledge Base

Per-guild documentation with revision history. Markdown editor, hierarchical pages, never lose institutional knowledge.

Unique to Valhalla
📊

Native Polls

Multi-select polls with expiry and live results. No bots needed.

Unique to Valhalla
🛡️

Roles & Permissions

53-bit hierarchical permission engine with channel overrides. Fine-grained control over who can do what, where.

🔐

SSO / SAML

Enterprise Single Sign-On with SAML 2.0 and OIDC. Connect to Okta, Azure AD, or any identity provider.

Enterprise
📜

Compliance

Message retention policies, legal holds, and audit exports. Meet regulatory requirements without third-party tools.

Enterprise
🤖

Bot API

Build custom integrations with applications, slash commands, and webhooks. Full API access for automation.

🔍

Full-Text Search

Lightning-fast message search powered by Meilisearch. Filter by author, channel, date, and more.

🏠

Self-Hostable

One-command deployment with Docker Compose. Your data, your server, your rules. Auto-TLS via Caddy.

Cross-Platform

Up and running in minutes

Self-host Valhalla with a single command.

# Clone the repository
git clone https://dgit01p.infra.ip413.de/bot_athene/valhalla.git valhalla
cd valhalla/deployments

# Run the setup script (generates secrets, starts everything)
./setup.sh

# That's it. Open https://your-domain in your browser.
